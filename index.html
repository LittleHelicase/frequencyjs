Simple to use Generators


// use a generator to create a signal
var sineGen = fjs.sine({frequency:200});
var signal = sineGen.create({length:100, samplingRate:400});

// use a generator to create a signal from multiple frequencies
var gen = fjs.sines([{frequency:10, amplitude:20},{frequency:30,amplitude:10}]);
var signal2 = gen.create({length:500, samplingRate:100});

Simply create signals of different frequencies. There is more to come...
